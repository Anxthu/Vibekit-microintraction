# VibeKit

**Motion & Microinteraction Library for Modern Web Applications.**

[**üöÄ Live Demo**](https://Anxthu.github.io/Vibekit-microintraction/)

VibeKit is a collection of high-quality, interactive, and customizable motion components designed to elevate the user experience of your web projects. By integrating smooth animations and responsive micro-interactions, VibeKit helps bridge the gap between static interfaces and dynamic, engaging user journeys.

![VibeKit Hero](public/hero-image.png)

## ‚ú® UI/UX Focus

VibeKit isn't just about "flashy" animations; it's about **usability** and **delight**.

-   **Feedback & Affordance**: Components like **Lift** and **Glow** provide immediate visual feedback, signaling interactivity and rewarding user actions.
-   **Error Handling**: The **Shake** animation offers a clear, intuitive, and non-intrusive way to communicate errors or invalid states.
-   **Visual Hierarchy**: **3D Tilt** and **Pulse** effects guide user attention to key elements, establishing a natural flow and hierarchy.
-   **Fluidity**: Built with **Lenis** for smooth scrolling and **Framer Motion** for seamless transitions, ensuring the application feels polished and responsive.

## üõ†Ô∏è Built With (DevX)

Designed for modern developers, VibeKit leverages the best in the React ecosystem for performance and ease of use.

-   **[React 19](https://react.dev/)**: Built on the latest React standards for robust component architecture.

## üì¶ Features

-   **Lift**: Elevates elements on hover with smooth spring physics, perfect for cards and buttons.
-   **Glow**: Adds a dynamic, cursor-aware glow effect to highlight active areas.
-   **Shake**: Physics-based wiggle animations for error states, notifications, or playful interactions.
-   **Pulse**: Rhythmic breathing effects to indicate urgency, activity, or live status.
-   **3D Tilt**: Premium, hardware-accelerated 3D cursor-following tilt effect for immersive cards.
-   **Morph**: Smooth shape transformations for fluid state changes.
-   **Retro 3D Hero**: A procedural, vaporwave-aesthetic 3D landscape generator.

## üöÄ Installation & Usage

Clone the repository and install dependencies to get started.

```bash
# Clone the repository
git clone https://github.com/Anxthu/Vibekit-microintraction.git

# Navigate to project directory
cd Vibekit-microintraction

# Install dependencies
npm install

# Start development server
npm run dev
```
