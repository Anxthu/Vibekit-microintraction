# VibeKit

**Motion & Microinteraction Library for Modern Web Applications.**

VibeKit is a collection of high-quality, interactive, and customizable motion components designed to bring your web projects to life. Built with **React**, **Framer Motion**, and **React Three Fiber**.

![VibeKit Hero](public/hero-image.png)

## Features

- **Lift**: Elevates elements on hover with smooth spring physics.
- **Glow**: Adds a dynamic, customizable glow effect.
- **Shake**: Attention-grabbing wiggle animations for error states or notifications.
- **Pulse**: Rhythmic beating effects for urgency or activity.
- **3D Tilt**: Premium 3D cursor-following tilt effect.
- **Morph**: Smooth shape transformations.
- **Retro 3D Hero**: A procedural Commodore 64-style 3D element.

## Tech Stack

- **React**: UI Framework
- **Framer Motion**: Animation Library
- **React Three Fiber**: 3D Rendering
- **Lenis**: Smooth Scrolling

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```
